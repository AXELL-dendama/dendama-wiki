Login方法¶

電玉のアプリはios/android以外からのアクセスの場合HPへリダイレクトする様設定してある
chrome等のエミュレーターでUAを偽装してもリダイレクトされることはないが、その先へは進めない。
また、nativeのバージョンチェックも行なっているので、エミュレータのUAを下記に変更する
DDAVの後のバージョンを最新にすること

Mozilla/5.0 (iPhone; CPU iPhone OS 11_0_3 like Mac OS X) AppleWebKit/604.1.38 (KHTML, like Gecko) Mobile/15A432 DDOS/iOS;DDOV/11.0.3;DDOM/iPhone;DDAV/1.6;
テストでPC等からログインする必要がある場合は下記をconsoleから実行する
  • prd
    found_uuid('00e20889-27d2-4606-b4e0-fd06954f383f')
  • dev
    found_uuid('05e897a8-01e2-4533-80f5-de5f26780bcd')

新規にユーザを作成する場合は下記を実行する(prdでは新規作成はなるべく行わないでください）

not_found_uuid()
