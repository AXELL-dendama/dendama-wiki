技名、技情報、3Dモデル

  • trick.js ：JavaScript リソース（Final trick, Series trick）
  • combo.js ：JavaScript リソース（組合せ技）
    • 各要素のプロパティ
      • level
        • 技のレベル（1 〜 16）
      • v2
        • 電玉V2のみで判定できる場合 true
      • single
        • クエスト等で単独で提示すべき技
        • secondTrick として判定される
      • supporting
        • 構え技、途中技、収め技、持替技のいずれかの場合 true
        • クエスト等で単独で提示すべきでない技
        • 組合せ技の一部として提示すべき技
      • hard
        • ジャイロの不具合により判定困難な技
        • 提示すべきでない技
      • time
        • 実施時間の目安
        • 単位は秒
        • 5秒間隔で設定してあります。
  • attitude.dae ：けん玉 3Dモデル
