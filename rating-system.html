How it looks in the database: 

{
    "TLvl": 
    4,
    "pastTG": 
    11,
    "pastAVGR": 
    "0",
    "pastRank": 
    "B-6-4",
    "classRank": 
    "B-6-4",
    "classLabel": 
    "amateur",
    "classLevel": 
    "50.00",
    "currentPPR": 
    "50",
    "currentRating": 
    "6.6"
}
            
                TLvl = current trick level (from 0 to 12)
                pastTG = how many elements (class levels) were used to get the PPR10
                pastRank = past rating (not currently used)
                pastRank = past rating code. The first digit is the "class" the second is the "currentRating" (which needs to be rounded) and the last is the level of trick.
                classRank = current rating code.
                classLabel = label to designate a user grade. Example: amateur, beginner, etc.
                classLevel = newest item in the history of logs ratings of the user.
                currentPPR = its the PPR result from the game used to create the rating game. This value is saved in the Classlevel (a field in the client) to make the history of that particular game.
                currentRating = current rating of the user.
