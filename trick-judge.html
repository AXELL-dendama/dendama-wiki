概要

技判定を行う Cocoa (Touch) Framework 、および Android Librry

リポジトリ

  • TrickJudge-Swift （Cocoa (Touch) Framework）
  • TrickJudge-Android （Android Library）

準備（iOS, macOS 編）

TrickJudge をリンクするアプリの Xcode プロジェクトのルートディレクトリを $(SRCROOT) とする。

  1. cd $(SRCROOT) を行う。
  2. $(SRCROOT)/Cartfile に以下内容を入れる。
  3. carthage update --platform iOS を行う。（もしくは --platform Mac 、以降 iOS で説明する。）
  4. $(SRCROOT)/Carthage/Build/iOS/TrickJudge.framework ができているのを確認する。
  5. TrickJudge をリンクするアプリの Xcode プロジェクトを開く。
  6. サイドバーの Project Navigator よりプロジェクトを選び、さらにターゲットを選ぶ。
  7. General > Link Frameworks and Libraries で +（追加）をクリックし、Add Other... より $(SRCROOT)/Carthage/Build/iOS/TrickJudge.framework を追加する。
  8. Build Phases で Run Script を追加し、Script として /usr/local/bin/carthage copy-frameworks と記述する。さらに、+（追加）をクリックし $(SRCROOT)/Carthage/Build/iOS/TrickJudge.framework を追加する。（ $(SRCROOT) は中身を展開する必要はなく、そのまま記述すればよい。）

準備（Android 編）

  1. TrickJudge-Android をダウンロード、もしくはチェックアウトする。
    • TrickJudge-Android プロジェクトのルートディレクトリを $(LIBROOT) とする。
  2. TrickJudge をリンクするアプリの Android Studio プロジェクトを開く。
  3. settings.gradle を以下のように修正する。
    include ':app'
    ↓ 
    include ':app', ':trickjudge'
project(':trickjudge').projectDir = new File('$(LIBROOT)/trickjudge')
    $(LIBROOT) の中身は展開しておく（例：'../TrickJudge-Android/trickjudge'）
  4. メニューから File > Project Structure... を開く。
  5. Modules の app を選択し、Dependencies タブを開く。
  6. + をクリックして、Module dependency から :trickjudge を追加する。

SensorData

Swift

open class SensorData {
    public init(buf: DataBuffer)
}

Java

public class SensorData {
    public SensorData(DataBuffer buf)
}

JudgeContext

技判定コンテキスト

Swift

open class JudgeContext {
    public init!()
    public init!(stillSeconds: Int)

    open func judge(_ items: [SensorData]) -> [StateResult]
    open func judge(_ item: SensorData) -> StateResult?
}

Java

public class JudgeContext {
    public JudgeContext() throws Exception
    public JudgeContext(int stillSeconds) throws Exception

    public List<StateResult> judge(List<SensorData> items)
    public StateResult judge(SensorData item)
}
