DENDAMAMA WIKI: documentation, reference code and development

ATTENTION:

This documentation is for references for other similar projects. Feel free to check and use the code/ideas.

The original version 1 and 2 documentation were written in Japanese and some of them were merged with this documentation, which mainly covers version 3.

The original version 1 and 2 were made using Ruby on Rails with jQuery and WebView as the base for the whole structure using AWS cloud services.

In this latest version, version 3, we moved the whole backend to use Python and Django for all the APIs to connect with the UI (still made in jQuery) and all other features of the mobile app.

The mobile app source is not explained here. The mobile app was made using Java (for Android) and Swift (for iPhone), which used custom firmwares made by DENDAMAMA inc. and connected all the features of the mobile to this architecture, which mainly uses WebView.
